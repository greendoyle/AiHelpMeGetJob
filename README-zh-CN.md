## 🇨🇳 中文说明

### 项目简介
这是一个基于无障碍服务的 Android 应用，智能分析职位描述和要求, 为求职者匹配岗位并进行初次沟通。

### 核心功能
- **无障碍解析**：通过 Android 无障碍服务读取 Boss 直聘应用界面
- **职位提取**：自动识别并提取职位卡片中的关键信息
- **AI分析**：LLM Agent智能分析职位匹配度, 进行初步沟通
- **自动化**：一键启动，可快速进行多个岗位的沟通


### 使用步骤
   - 打开本应用, 设置好相关参数, 包括求职相关的信息, 个人的简历信息, 以及LLM 的API KEY/URL的相关信息
   - 首页启动任务, App会检测是否开启无障碍模式, 如果未开启, 会引导至无障碍模式开启界面, 开启后方可开始启动任务
   - 启动任务后, 自动启动 Boss 直聘应用, 开始检测岗位和自动沟通


### 注意事项
- ⚠️ 必须在无障碍模式下运行
- ⚠️ 务必设置好所有参数再启动任务, 避免出现非法场景
- ⚠️ LLM API提供了测试按钮, 务必填写完以后点击测试, 确保API可用
- ⚠️ 本应用仅用于学习和交流, 请勿用于非法用途
